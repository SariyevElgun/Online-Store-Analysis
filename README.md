## Overview

The ** Online Store Sales Dashboard** provides an interactive overview of sales performance across outlets, item categories, outlet sizes, locations, and types. It is designed to help stakeholders quickly understand key metrics, trends over time, and category-level performance.

## Key KPIs

* **Total Sales:** $507.90K
* **Average Sales:** $140
* **Number of Items:** 3,631
* **Average Rating:** 3.9

These KPIs update dynamically based on selected filters.

## Filters Panel

Located on the left side of the dashboard, the filter panel allows users to refine the analysis:

* **Outlet Location Type:** All / Tier-based locations
* **Outlet Size:** Small / Medium / Large
* **Item Type:** Product categories (e.g., Fruits, Snacks, Dairy, etc.)

All visualizations respond instantly to filter changes.

## Dashboard Visuals

### 1. Outlet Establishment (Time Series)

* Displays total sales trend by year.
* Highlights growth and decline patterns across establishment years.

### 2. Fat Content Analysis

* **Donut Chart:** Sales split between *Low Fat* and *Regular* items.
* **Fat by Outlet:** Comparison of fat content sales across Tier 1, Tier 2, and Tier 3 outlets.

### 3. Item Type Performance

* Horizontal bar chart showing total sales by item category.
* Top-performing categories include Fruits, Snacks, and Household items.

### 4. Outlet Size Distribution

* Donut chart representing sales contribution by outlet size.
* Medium-sized outlets dominate total sales.

### 5. Outlet Location Analysis

* Sales distribution across Tier 1, Tier 2, and Tier 3 locations.
* Tier 3 locations contribute the highest share of sales.

### 6. Outlet Type Comparison

A detailed table comparing outlet types:

* **Outlet Type** (Grocery Store, Supermarket Type 1/2/3)
* **Total Sales**
* **Number of Items**
* **Average Sales**
* **Average Rating**
* **Item Visibility**
